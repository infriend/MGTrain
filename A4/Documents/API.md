## 2. 接口定义
### 2.1 通用说明
* 获取页面的接口直接返回页面
* 获取数据的接口前后端使用json交互，json数据格式定义如下。

| 字段        | 字段含义           | 类型  |  取值范围  |  是否可能为空  | 备注 |
| ------------- |:-------------:|:-----:|:-----:|:-----:|:-----:|
| success      | 成功与否的状态 | String |    |  |  |
| message      | 错误消息      | String |    |  |  |
| data | 返回的业务数据      | String |    |  |  |
